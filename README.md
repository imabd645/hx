# MaazX Website

A modern, light-themed website showcasing the MaazX autonomous engineering agent and all its functionalities.

## Features

- **Modern Design**: Clean, light-themed interface with responsive design
- **Complete Feature Showcase**: Detailed presentation of all MaazX capabilities
- **Tool Reference**: Comprehensive documentation of all available tools
- **Interactive Elements**: Code windows, terminal simulations, and hover effects
- **Mobile Responsive**: Works perfectly on all device sizes

## Project Structure

```
MaazX/
├── index.html          # Main HTML file
├── css/
│   └── style.css      # All styles
├── js/
│   └── script.js      # Interactive functionality
├── README.md          # This file
└── favicon.ico        # Website icon
```

## Sections

1. **Hero Section** - Introduction to MaazX with interactive code window
2. **Features** - Core capabilities and operating standards
3. **Tools** - Complete tool reference organized by category
4. **About** - Information about MaazX and its creator
5. **GitHub** - Open source information and repository links

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Poppins, Roboto Mono)
